CATEGORY: OPERATING SYSTEMS | WINDOWS 7

"My drives are missing on Windows Explorer"

Change the Windows Explorer target path

If you are using Windows 7 and Windows Explorer is not showing all of your folders and drives, you may need to fix the target path.

  1. Open the Windows Explorer Properties
  2. Select the Shortcut tab
  3. Set the Target path to: %windir%\explorer.exe

CATEGORY: OPERATING SYSTEMS | WINDOWS 10

"How can I set File Explorer to open to "This PC" instead of "Quick Access?"

Edit Folder Options

Open File Explorer and edit the General settings

  1. Open File Explorer
  2. Select File at the top left
  3. Select Change folder and search options
  4. Set Open File Explorer to: This PC

CATEGORY: OPERATING SYSTEMS

"I need access to a network drive"

Step 1. Set the Security permissions

If a user needs access to a specific folder, you may need to configure the Security Properties to give the user access to the contents in that folder before mapping the network drive.

  1. Right-click the folder the user needs access to
  2. Select Properties and go to the Security tab
  3. click Edit... and add the user/group
  4. Check all appropriate Allow checkboxes to grant the user/group access to the folder

Step 2. Map the network drive using File Explorer

After configuring the Security permissions, manually map the network drive on the user's computer.

  1. Open File Explorer
  2. right-click This PC and select Map network drive…
  3. In the Folder box type the folder path like this: \\server\share

Make sure you can ping the server

Use the IP address of the server if the hostname isn’t working

CATEGORY: OPERATING SYSTEMS | WINDOWS 10

Error: "You can't connect to the file share because it's not secure."

Enable SMBv1

SMBv1 is an old and insecure network file sharing protocol that is disabled on Windows 10 by default.

  1. Open Control Panel
  2. select Programs and Features
  3. Check SMB 1.0/CIFS File Sharing Support

CATEGORY: PRINTERS

How to add a Printer by IP Address

Step 1. Add the printer from the Control Panel

Before you can add the printer to the client's desktop, make sure you know the IPv4 address of the printer or print a configuration page. If you are helping the client remotely, assist them on how to print a configuration page from the printer.

  1. Open Control Panel and select Devices and Printers
  2. click Add a printer
  3. select "The printer that I want isn’t listed"
  4. select Add a printer using a TCP/IP address or hostname
  5. Enter the IPv4 address and select the correct driver on the next page

Step 2. Print a test page

It's best practice to make sure the printer is printing correctly before ending your session with the client.

  1. Open Control Panel and select Devices and Printers
  2. Right-click the printer and select Printer Properties
  3. Select Print Test Page

CATEGORY: PRINTERS

"My printer is randomly printing pages"

Clear the Print Spooler

If a printer is randomly printing pages on it's own, try clearing the Print Spooler.

  1. Press Winkey + r and type services.msc
  2. right-click Print Spooler select Stop
  3. Next, navigate to C:\Windows\System32\spool\PRINTERS
  4. Delete all the files in that folder and restart the Print Spooler

CATEGORY: PRINTERS

"My printer is printing a crazy symbols!"

Install the latest driver

If a printer is printing hieroglyphics it's usually a driver issue. Go to the manufacturer's website to install the latest driver. Every installation process is different. If you've installed the latest driver and the printer is still not printing correctly, make sure the printer is using the latest driver from the Control Panel.

  1. Open Control Panel and select Devices and Printers
  2. Right-click the printer and select Printer Properties
  3. Click the Advanced tab
  4. click on the Driver drop-down list and select the latest driver

CATEGORY: OUTLOOK

Error: "The set of folders cannot be opened."

Delete the .ost file and recreate the Outlook profile

If a client cannot open their Outlook because of this prompt, try deleting corrupt .ost file and recreating their Outlook Profile

  1. Go to C:\Users\username\AppData\Local\Microsoft\Outlook
  2. Delete the corrupt .ost file
  3. Right-click the printer and select Printer Properties
  4. Open Control Panel and select Mail
  5. Click Add... to create a new profile

CATEGORY: OUTLOOK

Error: "Library not registered."

Make changes to the Windows Registry

This error usually occurs after a upgrade. There may be an older key in the Windows Registry that needs to be deleted. Backup the key by saving it as an Export before making any changes to the Registry.

  1. Press Start, type regedit and click on Registry Editor
  2. Nagivate to HKEY_CLASSES_ROOT\Typelib\0006F062-0000-0000-C00-000000000046
  3. Backup the folder before deleting it

    4. If the registry has both 1.0 and 1.1 folders, delete the 1.0 folder

    If the registry has both 1.1 and 1.2 folders, delete the 1.2 folder

  4. Now try to open Outlook
  5. If that doesn't work, try running an Office repair

CATEGORY: OUTLOOK

Error: "Instant Search is not available when Outlook is running with administrator permissions."

Uncheck “Run As Administrator” for Outlook.exe

If Outlook Instant Search is not working properly, trying reconfiguring the Compatibility settings for Outlook.

  1. For 32-bit Windows go to C:\Program Files\Microsoft Office\Office14\

    2. For 64-bit Windows go to C:\Program Files (x86)\Microsoft Office\Office14\

  2. Right-click OUTLOOK.EXE and select properties
  3. Under the Compatibility tab, uncheck “Run As Administrator”

CATEGORY: POWERPOINT

"I can't open Powerpoint attachments from Outlook."

Edit Trust Center settings

If you are getting an error that says "If you trust the source of this presentation, click Repair." and clicking Repair does not fix the issue, try following these steps.

  1. Open Powerpoint, click on File and then Options
  2. select Trust Center and click on Trust Center settings...
  3. select Protected View and uncheck "Enable Protected View for files originating from the Internet"

CATEGORY: ACTIVE DIRECTORY

"I'm locked out of my desktop! I can't sign in."

Unlock the user's account on Active Directory

When a user's login attempts exceed the Account lockout threshold, they'll need an administrator to unlock their account.

  1. Log into the domain controller as an adminstrator and open Active Directory Users and Computers console
  2. Find the AD user, right-click and select Properties
  3. From the Account tab check the box next to Unlock Account and press OK

CATEGORY: EXCHANGE

How To Customize Mailbox Size

Customize the quota settings

If a client is unable to send or receive mail, it may be due to their mailbox being full. You can increase the size of their mailbox from the Exchange Admin Center.

  1. Log into the Exchange Admin Center, select Recipients and then Mailboxes
  2. Select the mailbox and go to Properties
  3. Select mailbox usage and check Customize the quota settings for this mailbox

    4. Issue a warning at (GB): sets the maximum storage limits before the user receives a warning

    Prohibit send at (GB): prevents the user from sending once the limit is reached

    Prohibit send and receive at (GB): prevents the user from sending or receiving email once the limit is reached.

CATEGORY: EXCHANGE

How To Create A Retention Policy

Step 1. Enable Archiving

Enable archiving for the mailboxes that will be assigned the retention policy. You can do this step first or save it for last. Note that when archiving is enabled, the “Default MRM Policy” is applied by default which includes the “Default 2 year move to archive” retention tag.

  1. Log into the Exchange Admin Center, select Recipients and then Mailboxes
  2. You can bulk-enable archiving by holding Shift or Ctrl
  3. On the right, look for Archive and select Enable

Step 2. Create A Retention Tag

When creating a new retention tag, it is applied to the entire mailbox by default.

  1. From Compliance Management navigate to Retention Tags and click Add+
  2. Name the retention tag and check a Retention action:

    3. Delete and Allow Recovery

    Permanently Delete

    Move to Archive

  3. Set the Retention period:

    4. Never or enter a number in days

Step 3. Create A Retention Policy

After creating the retention tag, create a retention policy and add the retention tag to that policy.

  1. From Compliance Management navigate to Retention Policies and click Add +
  2. Name the retention policy and add the retention tag you created in the previous step
  3. click Save

Step 4. Apply The Retention Policy

Apply the retention policy individually or by bulk

  1. Select a mailbox and select mailbox features
  2. Click on the drop-down list under Retention policy: and select a retention policy
  3. You can confirm that the retention policy is working by checking mailbox features and to the right it should say Archiving: Enabled
